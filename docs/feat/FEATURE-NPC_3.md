# Feature Spec: Dungeon Debugging, Character Leveling
---
## Requirements
---
- Add dungeon debugging tools (keybinds to be easily removed later):
    - Add a tool to kill all enemies on the floor
    - Add a tool to open all chests
    - Add a tool to skip to next floor

- Add global character leveling system (more info below)

### Global Character Leveling
---
In `ai/memory/player-profile.json`, add a global character level. Level up each time a user completes a quest for now just for simplicity
